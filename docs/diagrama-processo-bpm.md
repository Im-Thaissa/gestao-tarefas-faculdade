# Diagrama BPMN – Processo de Gestão de Tarefa

Este diagrama mostra o fluxo completo de criação, aceitação, execução, atualização e revisão de uma tarefa, com divisão clara entre os papéis de Solicitante e Responsável.

![Diagrama BPMN do Processo de Tarefa](img/diagrama-bpmn.png)

## Legenda

- **Solicitante:** Cria, define responsável, notifica, revisa e conclui a tarefa.
- **Responsável:** Consulta, executa, atualiza status, informa conclusão.

> Para edição do fluxograma, utilize a versão em BPMN abaixo (compatível com [bpmn.io](https://bpmn.io)).
